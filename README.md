# minify-file
Minify files in the browser environment

## Installation
Using npm:
``` Shell
npm install minify-file --save
```
Using yarn:
``` Shell
yarn add minify-file
```

## Features

### Minify images

## Examples

``` JavaScript
import minifyFile from 'minify-file'

minifyFile.minifyImage()
```
